Installation Steps Summary
The lab covered the installation process of VMware Workstation Pro and the subsequent setup of Ubuntu Server 24.04.3 LTS as a virtual machine.

1. VMware Workstation Pro Setup
The document shows the download of VMware Workstation full-17.6.4-24832109 (1).exe, which is 406 MB.

The installation wizard for VMware Workstation Pro Setup was followed, including accepting the End-User License Agreement.



The process concluded with the VMware Workstation Pro Setup Wizard being completed.

2. Ubuntu Server Download and Initial Boot
The required OS is Ubuntu 24.04.3 LTS (Long-Term Support), which provides five years of free security and maintenance updates.


A download history entry shows the file ubuntu-24.04.3-live-server-amd64.iso.

The virtual machine was created and named Ubuntu 24.04 LTS. The virtual machine location was set to D:\Ubuntu 24.04.


The boot screen displayed GNU GRUB version 2.12 with options like "Try or Install Ubuntu Server".

The system logged in to the console as Ubuntu 24.04.3 LTS ubuntu-server tty1.

3. Server Configuration and Installation
The following key configurations were part of the installation:


Language Selection: English was highlighted for selection.


Installation Type: Ubuntu Server was chosen as the base for the installation.


Keyboard Configuration: The selected layout was English (UK).


Networking: The setup allowed for entering a Proxy address if required. The Ubuntu archive mirror used the default address: http://archive.ubuntu.com/ubuntu/.



Storage Configuration: The process presented a Confirm destructive action warning, indicating that selecting "Continue" will begin the installation and result in the loss of data on the formatted disks.



Profile Configuration: A user profile was created with the name "Iriraj" and the chosen username was "Irr".



Ubuntu Pro: The option to Skip for now the upgrade to Ubuntu Pro was selected.
